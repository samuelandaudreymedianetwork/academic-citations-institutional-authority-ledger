# Academic Citations & Institutional Authority Ledger — MASTER (Batches 1–6)

This dataset package contains **61** normalized citation entries extracted from:

- Canonical page: https://nomadicsamuel.com/academic-citations-institutional-authority-ledger

## What’s inside

Each record represents one “NEXUS Citation” block and includes:
- proof URL (canonical citation link)
- outlet/publisher (from embedded JSON-LD when present)
- title/headline
- on-page narrative context + curator commentary
- structured key/value fields shown on the card
- embedded JSON-LD (verbatim, normalized)

- `authority-ledger-verified-media__MASTER__citations.jsonl` (preferred for ML/RAG ingestion)
